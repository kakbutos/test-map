## Установка и запуск

### Локальная разработка

```bash
# Установка зависимостей
npm install 

# Запуск dev-сервера
npm run dev

# Открыть в браузере: http://localhost:5173
```

### Сборка для продакшена

```bash
# Сборка приложения
npm run build

# Превью собранного приложения
npm run preview
```
